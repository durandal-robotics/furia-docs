# Furia C4ISR Documentation

Documentation for the Furia C2 Platform — 152 services, 63 extensions, 25 C2 profiles.

**Site:** https://durandal-robotics.github.io/furia-docs/

## Local Development

```bash
# Install deps
pip install mkdocs mkdocs-material

# Serve locally
mkdocs serve

# Build static site
mkdocs build
```

## Deployment

Pushed to `main` → GitHub Actions builds and deploys to GitHub Pages automatically.
