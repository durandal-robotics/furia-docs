# Glossary

| Term | Definition |
|------|------------|
| **BDA** | Battle Damage Assessment — evaluation of strike effectiveness |
| **BFT** | Blue Force Tracking — position and status of friendly units |
| **C2** | Command and Control — exercise of authority over assigned forces |
| **C4ISR** | Command, Control, Communications, Computers, Intelligence, Surveillance, Reconnaissance |
| **COA** | Course of Action — a plan of action to accomplish a mission |
| **COP** | Common Operating Picture — shared display of relevant information |
| **CoT** | Cursor on Target — XML-based message format for tactical data |
| **C-UAS** | Counter-Unmanned Aircraft Systems — defense against drones |
| **EMCON** | Emission Control — management of electronic emissions |
| **FRAGO** | Fragmentary Order — changes to an existing OPORD |
| **GCS** | Ground Control Station — control center for UAV operations |
| **IHL** | International Humanitarian Law — laws of armed conflict |
| **IPB** | Intelligence Preparation of the Battlefield |
| **ISR** | Intelligence, Surveillance, Reconnaissance |
| **MUM-T** | Manned-Unmanned Teaming — helicopter + UAV coordination |
| **OPORD** | Operation Order — directive for execution of a military operation |
| **PNT** | Positioning, Navigation, and Timing — GNSS, INS, fallback fusion |
| **ROE** | Rules of Engagement — directives governing use of force |
| **SAPIENT** | STANREC 4869 — C-UAS sensor/effector pub-sub protocol |
| **SitRep** | Situation Report — status update on current operations |
| **STANAG** | NATO Standardization Agreement |
| **TEWA** | Threat Evaluation and Weapon Assignment |
| **WARNORD** | Warning Order — preliminary notice of future operation |
